# BOTHREI PRINTERS Website

## 🖨️ Your One-Stop Digital Hub!

A modern, responsive website for BOTHREI PRINTERS, a printing and digital services business.

### 🎨 Color Scheme
- **Light Blue** (#4aa3ff) - Primary brand color (used for "BOTHREI" and accents)
- **Black** (#000000) - Secondary color (used for "PRINTERS" and text)

### 📋 Services Offered
- **Core Services**: Printing, Photocopying, Typing, Scanning
- **Additional Services**: Online Job Applications, Graphic Design, Internet Browsing
- **Other Services**: Lamination, Binding, Passport Photos, Fax

### 🚀 Features
- Fully responsive design
- Mobile-friendly navigation
- Smooth scrolling animations
- Contact form with validation
- Service cards with hover effects
- Pricing tables
- Business hours display
- Social media integration
- Scroll to top button
- Form notifications
